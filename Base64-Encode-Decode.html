What is Base64?

Base64 is a binary-to-text encoding scheme that represents binary data in an ASCII string format. It's commonly used when there is a need to encode binary data that needs to be stored and transferred over media designed to deal with text. Our free online tool allows you to easily encode text to Base64 or decode Base64 back to readable text. This tool works on all devices including smartphones, tablets, and desktop computers.

Common uses for Base64 include encoding email attachments, embedding binary data in JSON or XML, and encoding credentials for basic authentication in HTTP headers.
